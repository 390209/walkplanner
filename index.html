⚙️ Configuración Microsoft

Necesitás registrar una app en Azure:
1. Ve a portal.azure.com → App registrations
2. "New registration" → nombre: WalkPlanner
3. Redirect URI: tipo "SPA" → pegá la URL de esta página
4. En "API permissions" agregá: Tasks.ReadWrite, User.Read
5. Copiá el Application (client) ID
Tus datos nunca salen de tu dispositivo. Todo funciona con OAuth oficial de Microsoft.
